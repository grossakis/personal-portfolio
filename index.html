Hangman Game

Here is a hangman game I created. Players use the keyboard to guess letters and try and fill the blanks. Pretty straightforward. This one happens to have an animal theme. Overwatch theme coming soon!

Futurama Trivia Game

This is a trivia game I created based on the animated sitcom "Futurama". Players are asked 5 random questions. Each question is timed and results are displayed once all 5 questions are answered.

Meal Planner

Working with a group, I helped to create a meal planning app. Users can explore the Edemam recipe database and bookmark recipes they like. All the recipes that are bookmarked are stored in google's Firebase database and can be viewed along with nutritional value charts for effective and healthy meal planning.
